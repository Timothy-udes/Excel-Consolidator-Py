# Excel Consolidator Py

A Python script to consolidate multiple Excel files into a single file efficiently.

## Features
- Combines multiple Excel files in a folder into one.
- Handles multiple sheets if needed.
- Easy to customize for your Excel structure.
- Saves consolidated output automatically.

## Requirements
- Python 3.x
- pandas (`pip install pandas`)
- openpyxl (`pip install openpyxl`)

## Usage
1. Place all Excel files you want to consolidate in one folder.
2. Update the `folder_path` variable in `Consolidator.py` with your folder location.
3. Run the script:
```bash
python Consolidator.py
